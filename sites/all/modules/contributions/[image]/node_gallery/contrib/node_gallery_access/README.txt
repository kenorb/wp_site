DESCRIPTION
--------------------------
This is a contribution of the node gallery module. Use this module to control your node gallery's access.
Currently support three access types: public, private and password. 

REQUIREMENTS
---------------
- Node gallery module. 

INSTALLATION
---------------
- Enable the modules.
- Opening your gallery's edit form, you'll find an access setting fieldset under the body textarea.
