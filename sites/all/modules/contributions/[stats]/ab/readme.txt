        ------------------------------------------------------------------------------- 
        INSTALLATION
        ------------------------------------------------------------------------------- 
        
        Enable the module. Optionally install the open_source_chart_api module for 
        pretty charts :)
        
        ------------------------------------------------------------------------------- 
        PERMISSIONS
        ------------------------------------------------------------------------------- 
        
        administer ab 
          - Control the automated ApacheBench cron tasks.
        
        view ab reports
          - view reports from previous ab() invocations.
        
        ------------------------------------------------------------------------------- 
        PUBLIC API
        ------------------------------------------------------------------------------- 
        
        ab()
        ab_clear_all()
