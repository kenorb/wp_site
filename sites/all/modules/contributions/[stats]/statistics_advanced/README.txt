INSTALLATION
------------

See http://drupal.org/getting-started/5/install-contrib for instructions on
how to install or update Drupal modules.

Once Statistics Advanced Settings is installed and enabled, you can adjust the
advanced statistics settings at admin/reports/settings/advanced.
